# New-PveNodesQemuStatusShutdown

## SYNOPSIS

## SYNTAX

```
New-PveNodesQemuStatusShutdown [[-PveTicket] <PveTicket>] [[-Forcestop] <Boolean>] [[-Keepactive] <Boolean>]
 [-Node] <String> [[-Skiplock] <Boolean>] [[-Timeout] <Int32>] [-Vmid] <Int32>
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Shutdown virtual machine.
This is similar to pressing the power button on a physical machine.
This will send an ACPI event for the guest OS, which should then proceed to a clean shutdown.
