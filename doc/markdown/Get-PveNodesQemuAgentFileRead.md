# Get-PveNodesQemuAgentFileRead

## SYNOPSIS

## SYNTAX

```
Get-PveNodesQemuAgentFileRead [[-PveTicket] <PveTicket>] [[-Count] <Int32>] [[-Decode] <Boolean>]
 [-File] <String> [-Node] <String> [[-Offset] <Int32>] [-Vmid] <Int32> [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION
Reads the given file via guest agent.
Is limited to 16777216 bytes.
