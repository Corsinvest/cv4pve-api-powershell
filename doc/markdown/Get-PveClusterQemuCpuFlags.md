# Get-PveClusterQemuCpuFlags

## SYNOPSIS

## SYNTAX

```
Get-PveClusterQemuCpuFlags [[-PveTicket] <PveTicket>] [[-Accel] <String>] [[-Arch] <String>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
List of available CPU flags.
Currently only implemented for x86_64, returns an empty list for aarch64.
