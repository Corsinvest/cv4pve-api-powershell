### -ResourceMode
Controls how HA managed resources are handled while disarmed.
The current state of resources is not affected.
'freeze'':' new commands and state changes are not applied.
'ignore'':' resources are removed from HA tracking and can be managed as if they were not HA managed.
Enum: freeze,ignore
