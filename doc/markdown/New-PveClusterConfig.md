# New-PveClusterConfig

## SYNOPSIS

## SYNTAX

```
New-PveClusterConfig [[-PveTicket] <PveTicket>] [-Clustername] <String> [[-LinkN] <Hashtable>]
 [[-Nodeid] <Int32>] [[-Votes] <Int32>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Generate new cluster configuration.
If no links given, default to local IP address as link0.
